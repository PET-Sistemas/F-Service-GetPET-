
# F-Service [ GetPET ]

O projeto do GetPET consiste em uma aplicação web-mobile com o intuito de facilitar pequenos serviços em condominios residenciais. Sua principal funcionalidade é a comunicação entre os usuários que fornecem serviços e os que necessitam de tal, podendo ser ofertado valores, tipo, local, data e horario em que o trabalho será realizado.  A aplicação será feita por meio do paradigma orientado a objetos sendo realizado em linguagem JAVA.
